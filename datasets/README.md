## PubFig Dataset

We use PubFig dataset (about 187MB).
> Use `wget` to download.
> [Download Link](https://www.kaggle.com/datasets/kaustubhchaudhari/pubfig-dataset-256x256-jpg)
> unzip it and rename it `pubfig`
